Ativo

Portal de Notícias - Laravel + Vue.js

🎥 Demonstração do Projeto

Veja o portal em funcionamento com todas as funcionalidades implementadas

Aplicação full-stack moderna desenvolvida com arquitetura robusta e tecnologias de ponta. O projeto implementa um portal de notícias completo utilizando Laravel 11 como backend API, Vue.js 3 com Inertia.js para uma SPA híbrida, e Tailwind CSS para estilização responsiva.

A aplicação integra-se com a NewsAPI para busca de notícias em tempo real, implementa sistema de cache inteligente com Laravel Cache, utiliza SQL Server como banco de dados principal com índices otimizados para performance, e inclui funcionalidades avançadas como modo escuro automático, menu de acessibilidade completo e design responsivo.
